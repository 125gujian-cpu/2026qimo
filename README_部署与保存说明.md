# 部门法期末刷题 PWA

一个网址刷两个部门法：

- 行政法：110 题；
- 国际法：116 题；
- 两科题库、进度、错题、收藏互相隔离；
- 支持章节练习、未做题、随机练习、错题复刷、收藏题；
- IndexedDB 自动保存；
- JSON 学习记录导出、导入；
- PWA 离线缓存，可添加到手机、iPad、电脑桌面。

## GitHub Pages 部署

将本目录里的全部文件上传到仓库根目录，然后在 `Settings -> Pages` 里选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/ (root)`。

同一固定网址下更新网站文件，通常不会清除 IndexedDB 学习记录。换设备、换浏览器或清除站点数据前，请先在“数据”页导出 JSON 备份。
